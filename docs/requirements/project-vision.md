# AI Interview & Viva Simulator

## Problem Statement

Many students and job seekers possess technical knowledge but struggle to communicate confidently during interviews and viva examinations.

Traditional preparation methods provide limited real-time feedback on communication quality, confidence, answer relevance, and interview readiness.

The goal of this project is to create an AI-powered platform that simulates realistic interview and viva environments and provides actionable feedback to help users improve.

---

## Vision

Build an AI-powered interview and viva simulation platform that helps users transform knowledge into communication confidence.

---

## Target Users

- College students
- Freshers preparing for placements
- Job seekers
- Professionals
- Non-native English speakers

---

## Core Features

### Interview Mode

Users can:

- Select a role
- Select difficulty
- Participate in an AI interview
- Receive personalized feedback

### Viva Mode

Users can:

- Select a subject
- Select a topic
- Answer viva questions
- Receive topic-specific evaluation
